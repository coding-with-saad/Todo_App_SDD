# Quickstart: Todo In-Memory Python CLI

## Prerequisites
- Python 3.12+

## Run Application
```bash
python -m src.app
```

## Running Tests
```bash
python -m unittest discover tests/unit
```

## Interactive Session Example
1. Run `python -m src.app`.
2. Select "Add Task" from the menu.
3. Enter task title.
4. Select "View Tasks" to see the list with IDs.
5. Select "Mark Complete" and enter the ID.
6. Select "View Tasks" again to verify the status change.
7. Select "Exit" to close (Note: Data is lost on exit in Phase I).
